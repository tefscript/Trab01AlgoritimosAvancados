Este repositório contém a implementação dos problemas propostos na atividade de revisão de estruturas de dados, desenvolvidos em **Python** e organizados nas categorias:

- **Arrays**
- **Listas Encadeadas**
- **Pilhas**

Cada arquivo possui testes simples no final do código para facilitar a execução e a verificação dos resultados.

---

## Objetivo do projeto

O objetivo deste projeto é reunir as soluções dos problemas da atividade, mantendo a organização por tema e permitindo que qualquer pessoa consiga executar os arquivos individualmente pelo terminal.
